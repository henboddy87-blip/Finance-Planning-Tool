# Personal Finance Dashboard

A comprehensive personal finance management dashboard built with React and Tailwind CSS. This tool allows users to track their net worth, manage budgets, plan investments, track loans, and generate professional financial reports.

## Features

- **Dashboard**: Comprehensive overview of financial health with charts and metrics.
- **Net Worth Tracking**: Track assets and liabilities with historical trends.
- **Budgeting**: Create and monitor budgets with visual tracking.
- **Investment Tracker**: Manage stocks, ETFs, and crypto holdings with performance tracking.
- **Goals**: Set and track progress towards financial goals.
- **Retirement Planning**: Plan for retirement with projection tools.
- **Debt Management**: Track and plan debt repayment strategies (Snowball vs. Avalanche).
- **Calculators**: Includes tools like Loan Calculator and Compound Interest Calculator.
- **Reports & Exporting**: Generate and export financial reports in various formats (PDF, Excel, JSON).
- **Authentication**: Secure login and registration system.
- **Theme**: Light and Dark mode support.
- **Localization**: English and Khmer language support.

## Tech Stack

- **Framework**: React
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide 

## Getting Started

-npm run dev for run program

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd "Personal Finance Dashboard"
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Usage

Run the development server:

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.
